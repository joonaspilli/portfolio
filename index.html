Linear / Dixu

In my current position as a full stack developer at Linear I have helped to deliver several key features to enhance and modernize the working methods of real estate professionals. My work has included designing and building many feature-rich components and flows with complex requirements while closely meeting predetermined responsive visual designs. I have also created several API-endpoints, including ones that deal with demanding image processing.

All of my contributions to Linear's web services are made to the web applications served to their B2B customers, of which some are only available by registering on Linear's website. You can still glimpse the offered services without registration, though.

In addition to my work with Linear I have helped to deliver the renewed web services of Linear's subsidiary company Dixu where my work has mostly focused on implementing various UI enhancements. For some examples of my work, checkout Dixu's front page hero area and listing page's custom image/video carousel.

Rush Knight

For the final project of a game development course at JAMK I created an HTML5 browser game with the Phaser framework called Rush Knight. This was my first proper introduction to working with JavaScript and I also consider it as my first "real" software development project. I spent more time working on this game than I care to admit, but with all the designing, programming, animating, composing and testing it still remains as the most fun project that I've ever done, really. Here's hoping I get inspired to do another game project soon.

If you're gonna go ahead and check out the code, don't expect it to be marvelous or a good representation of my current ability, I've come a long way since.

Accessible SPA

I started working on my Bachelor's Degree thesis while still working at Frantic where web accessibility was an important part of everyday work. I came to find web accessibility an intriguing subject and it's importance increasingly more relevant, so making my thesis about accessibility in modern web development was an obvious choice.

While it's not exactly a feast for eyes, the single-page application built for the purposes of my thesis manages to showcase a simplified example of how modern single-page web applications can be done in accordance with W3C's WCAG 2.1. If you're up for it, test it out with a worthwhile screen reader software and you should expect a fairly smooth experience, considering the circumstances, anyway.
