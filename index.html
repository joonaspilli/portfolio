Linear / Dixu

In my current position as a full stack developer at Linear I have helped to deliver several new key features to enhance and modernize the working methods of real estate professionals. My work has included building many feature-rich components and order flows with complex requirements, which I have successfully delivered in a user-friendly experience. I have also created many API endpoints, including ones that deal with demanding image processing. Practically all of my contributions to Linear's web services are to the web applications served for their B2B customers, so feel free to explore the offered services on Linear's website .

In addition to my work with Linear, I have also helped to deliver the renewed web services of Linear's subsidiary company Dixu . With Dixu my work has mostly been with building reusable user interface components and forms. The front page hero section and the listing page's image + video carousel are some of the a bit larger tasks that I've done for Dixu.

Frantic / Finnair

During my time as a junior developer at Frantic I created several high profile campaign landing pages for the Finland's largest airline Finnair . I also contributed a lot to the development of Finnair's renewed flight offers page , which I consider possibly the most interesting project of my professional career, from the working perspective at least.

To boost the productivity of everyday work I also helped to improve some of the internal tools and working methods of Frantic. I also built many custom tools to speed up some of the more "tedious" yet important work. Thanks these efforts, I received positive feedback for the speed and quality of my work several times.

Rush Knight

For the final project of a game development course at JAMK I created an HTML5 browser game with the Phaser framework called Rush Knight . This was my first proper introduction to working with JavaScript and I also consider it as my first "real" software development project. I spent more time working on it than I care to admit, but it still remains as the most fun project that I've done. Here's hoping I get inspired to do another game project soon.

Accessible SPA

I started working on my Bachelor's Degree thesis while I was still employed at Frantic , where web accessibility was a fairly important part of everyday work. I came to find web accessibility an intriguing subject and it's importance more relevant than ever before, so making my thesis about accessibility in modern web development was only natural.

While it's not exactly a pleasure for the eyes, the single-page application built for the purposes of my thesis manages to showcase a simplified example of how modern single-page web applications can be done in accordance with W3C's WCAG 2.1.
